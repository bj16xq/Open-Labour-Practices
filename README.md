# Open Labour Union Practices
This repository hosts information on and files for open union practices (OUP) and open union resources (OUR).
The purpose behind OUP/OUR is to promote collaboration, inclusion, and innovation in union activities by sharing effective resources that are updated as improvements are made. This idea extends from the tradition of open educational practices (OEP)

Templates are available in Open Document Format (ODF) which is non-proprietary but mostly compatible with proprietary platforms such as Microsoft Word.

The following resources are currently available:
1. [Annual General Meeting agenda template](https://github.com/bj16xq/Open-Union-Practices/blob/main/Union%20AGM%20Agenda%20Package%20Template%20v1.2.md)
2. [Executive Meeting agenda template](https://github.com/bj16xq/Open-Union-Practices/blob/main/Union%20Executive%20Meeting%20Agenda%20Package%20Template%20v1.1.md)
3. Executive Reflection guide template
4. [JHSC Representative Reflection guide](https://github.com/bj16xq/Open-Union-Practices/blob/main/JHSC%20Rep%20Reflection%20Prompts%20v1.1.md)
